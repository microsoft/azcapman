## Goal

Verify the new ISF ratio blob CSV data source is accessible and has the expected schema for quota script consumption.

## Why

3. `ArmSkuName` column contains VM SKU values matching `Standard_*` pattern
4. CSV contains SQL-related SKUs (confirming the `-notlike "*sql*"` filter is still needed)
