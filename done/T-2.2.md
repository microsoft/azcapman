# T-2.2 — Implement catalog data fetcher for Az PowerShell scripts

## Goal

Create a `Get-VMCatalogData` function that calls the Azure Reservations Catalog REST API via `Invoke-AzRestMethod` with pagination support. This function replaces the CSV download as the data source for VM SKU discovery.

## Why

Per [Microsoft docs](https://learn.microsoft.com/en-us/azure/cost-management-billing/reservations/instance-size-flexibility#extract-instance-size-flexibility-ratios-using-azure-catalogs-api), the Azure Catalogs API is the authoritative source for ISF ratio data. Using `Invoke-AzRestMethod` (part of `Az.Accounts`, already required) avoids adding `Az.Reservations` as a new dependency.

## Files

- `scripts/quota/Get-AzVMQuotaUsage.ps1`
- `scripts/quota/Show-AzVMQuotaReport.ps1`

## API details

Endpoint: `GET /subscriptions/{subscriptionId}/providers/Microsoft.Capacity/catalogs?api-version=2022-11-01&$filter=ReservedResourceType eq 'VirtualMachines' and Location eq '{location}'`

Response shape:
```json
{
  "value": [
    {
      "resourceType": "virtualMachines",
      "name": "Standard_D2s_v5",
      "skuProperties": [
        { "name": "ReservationsAutofitGroup", "value": "Ddsv5 Series" },
        { "name": "ReservationsAutofitRatio", "value": "2" }
      ]
    }
  ],
  "nextLink": "https://..."
}
```

## Acceptance criteria

1. Function `Get-VMCatalogData` is added to both Az PowerShell scripts
2. Uses `Invoke-AzRestMethod` — no `Az.Reservations` module dependency
3. Accepts `$SubscriptionId` and `$Location` parameters
4. Implements pagination via `nextLink` (do/while loop)
5. Returns array of `[PSCustomObject]@{ InstanceSizeFlexibilityGroup; ArmSkuName; Ratio }`
6. Extracts `ArmSkuName` from `.Name`, group from `ReservationsAutofitGroup` SkuProperty, ratio from `ReservationsAutofitRatio` SkuProperty
7. Handles items missing ISF properties gracefully (skip, don't error)
8. Function includes comment-based help (`.SYNOPSIS`, `.DESCRIPTION`, `.PARAMETER`, `.OUTPUTS`)
9. Both scripts parse without syntax errors after the change
